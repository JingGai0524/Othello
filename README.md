# Othello

这里是Sustech java1 final project 黑白棋项目

本项目基于java swing实现全部UI界面（中世纪Viking风格）

支持本机的人人对战和人机对战

人机对战中的AI实现借鉴了蒙特卡洛树搜索和信心上限算法

可读档存档 可悔棋 可重新开始对局 可回放对战过程 可开启上帝模式

如有问题 请在issue中提出
